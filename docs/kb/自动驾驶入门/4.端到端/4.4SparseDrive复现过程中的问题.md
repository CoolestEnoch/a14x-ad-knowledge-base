# 4.4 SparseDrive复现过程中的问题

1. mmcv一定要安装mmcv-full 版本，建议使用cuda113+torch1.11.0, mmcv安装1.7.2 版本。使用以下命令: pip install mmcv-full -f [https://download.openmmlab.com/mmcv/dist/cu113/torch1.11.0/index.html](https://download.openmmlab.com/mmcv/dist/cu113/torch1.11.0/index.html) -i [https://pypi.tuna.tsinghua.edu.cn/simple](https://pypi.tuna.tsinghua.edu.cn/simple)
2. flash-attn记得离线安装，flash_attn-0.2.2+cu113torch1.11.0-cp38-cp38-linux_x86_64.whl 下载地址是[https://github.com/Dao-AILab/flash-attention/releases](https://github.com/Dao-AILab/flash-attention/releases)。将文件移动到sparsedrive文件夹下，执行pip install flash_attn-0.2.2+cu113torch1.11.0-cp38-cp38-linux_x86_64.whl 即可
3. 其他环境安装按照sparsedrive 中的设定即可。
