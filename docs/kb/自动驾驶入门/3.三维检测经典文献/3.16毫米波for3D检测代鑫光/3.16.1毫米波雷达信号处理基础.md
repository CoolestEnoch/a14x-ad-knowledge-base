## 一、毫米波雷达的概念及核心原理
雷达(Radar)-**RA**dio **D**etection **A**nd **R**anging: 利用无线电对目标进行探测和测距。

**雷达与其它传感器的比较优势和特点：**

+ 全天候全天时，解决了光学传感器在夜间和雨天无法正常工作的问题
+ 分辨率高，相对激光雷达具有成本优势，有望替代激光雷达实现雷达点云成像
+ 作用距离远，应用场景广泛，是未来高级自动驾驶应用的核心技术
+ 频率越高体积越小，77GHz天线尺寸是24GHz天线尺寸的1/3
+ 更容易实现单芯片集成结构
+ 更高的分辨率和探测精度（距离、角度和速度）

**雷达的劣势：**

+ 角度分辨率低于激光雷达
+ 易受金属反射干扰
+ 点云稀疏，难以实现高精度目标识别和分类
+ 对非金属目标的探测能力较弱，小目标（如行人、动物、锥桶）可能无法被准确探测
+ 毫米波雷达随着天线数量的增加，数据量急剧增大

### 1.雷达测距
#### 原理
$ R = \frac{c \cdot \tau}{2} $

其中，$ R $ 是目标距离，$ c $ 是电磁波在空气中的传播速度(光速)，$ \tau $ 是电磁波往返目标的时间。仅依靠时延$ \tau $与光速$ c $ 关联，即可直接解算目标距离，无需拍频参与，这是测距的本质原理。

#### FMCW雷达工程实现(拍频)
FMCW（Frequency Modulated Continuous Wave）雷达不直接测量微小时延，而是通过发射频率连续变化的信号（Chirp信号），将时延差转化为频率差（拍频），间接实现测距，具体如下：

+ FMCW雷达按PRT（起始到起始）周期工作，一个周期内先发射一整段线性调频Chirp波（频率从低到高匀速扫频），发射完毕后进入静默等待期，不发射信号，等待回波抵达并处理。
+ 即使发射Chirp的过程中回波未返回，静默等待期内接收的回波，依旧可与本次发射的整段调频波形做整体频率比对，计算出拍频。
+ 拍频$ f_b $与目标距离$ R $的关系为：

$ f_b = \frac{2B R}{c T} $

+ 其中，$ B $ 是雷达信号的带宽，$ T $ 是Chirp信号的持续时间，$ c $ 是电磁波在空气中的传播速度(光速)。通过测量拍频$ f_b $，可以计算出目标距离$ R $。

> **关键补充：**  
拍频与时延相关，用于测距；多普勒频移与目标相对运动相关，用于测速，二者概念不同，不可等同。  
理论上仅静止无相对运动的目标，不会产生多普勒频率偏移；  
但FMCW雷达发射频率持续线性变化，回波因传播存在往返时延，接收时刻雷达本地发射频率已发生改变，因此自发波与回波对比依旧会生频率差，该频率差即为拍频，依靠拍频即可完成静止目标测距。
>

> **回波时序说明：**  
对于同一距离单一目标，按低频到高频顺序发射信号，反射回波依旧遵循发射时序，低频回波先抵达，高频回波后抵达；只有场景中同时存在远近不同多个目标时，远距离目标的早期低频回波，才会晚于近距离目标的后期高频回波到达接收端，出现回波时序错乱（并非同目标回波乱序）。
>

#### 距离分辨率
距离分辨率$ \Delta R $是雷达能够区分两个目标的最小距离，计算公式为：

$ \Delta R = \frac{c}{2B} $

其中，$ B $ 是雷达信号的带宽。带宽越大，距离分辨率越高。

### 2.雷达测速
雷达依托多普勒频移实现直接测速。

> 拍频与时延相关，用于测距，多普勒频移与相对运动相关，用于测速，二者概念不同。  
当目标与雷达存在相对运动时，雷达接收的回波信号频率会发生偏移，该频率偏移量称为多普勒频率$ f_d $，它与目标相对速度呈线性对应关系，测速公式为：
>

$ f_d = \frac{2v}{\lambda} $

式中：$ f_d $ 为多普勒频率，$ v $ 为目标相对雷达的径向运动速度，$ \lambda $ 为雷达发射电磁波的波长，$ f $ 为发射波频率，反射波频率$ f'=f+f_d $，反射波波长$ \lambda'=\frac{c}{f'} $。  
当目标接近雷达时，$ f_d $ 为正值，$ \lambda' < \lambda $；  
当目标远离雷达时，$ f_d $ 为负值，$ \lambda' > \lambda $。

> 补充：运动目标的回波会同时叠加拍频（距离项）与多普勒频移（速度项），最终混频得到的总频差 = 拍频 + 多普勒频移，雷达通过算法分离两项频差，分别解算距离与速度。
>

#### 速度分辨率
速度分辨率$ \Delta v $是雷达能够区分两个目标的最小速度差，计算公式为：

$ \Delta v = \frac{c}{2fT} $

其中，$ f $ 是雷达发射波频率，$ T $ 是雷达信号的持续时间。持续时间越长，速度分辨率越高。

#### 多 PRF 调频与速度模糊
+ PRF（脉冲重复频率）：指雷达每秒发射Chirp信号的次数，定义为相邻两段Chirp信号“起始到起始”的周期，与发射载波频率是两个不同概念（载波频率是Chirp信号本身的频率，PRF是发射周期的频率）。脉冲重复时间PRT是PRF的倒数，即PRT = $ \frac{1}{PRF} $。
+ 速度模糊产生原因：当目标相对速度过快，多普勒频率超出PRF的检测范围，导致雷达无法准确识别速度方向和大小，出现速度模糊。
+ 三重频（多波形调频）：通过发射多种不同PRF的Chirp信号，结合多组回波数据进行解算，可有效消除速度模糊和距离模糊，提升测速、测距的准确性。

##### PRF与最大不模糊速度、最大不模糊距离的关系
相邻Chirp信号接收回波的相位差需满足$ |\Delta \phi| < \pi $，才能避免速度模糊，结合多普勒频移公式$ f_d = \frac{2v}{\lambda} $，可得：

$ |\Delta \phi| = 2\pi f_d \cdot \frac{1}{PRF} < \pi $

解出多普勒频移的最大可检测范围为$ -\frac{PRF}{2} \sim \frac{PRF}{2} $，即最大可检测多普勒频移$ f_{d,max} = \frac{PRF}{2} $；  
将$ f_{d,max} = \frac{PRF}{2} $代入多普勒频移公式$ f_d = \frac{2v}{\lambda} $，反推最大不模糊速度：

$ v_{max} = \frac{f_{d,max} \cdot \lambda}{2} = \frac{\frac{PRF}{2} \cdot \lambda}{2} = \frac{\lambda \cdot PRF}{4} $

总结：PRF越高，最大不模糊速度越大，但会导致最大不模糊距离减小，二者存在trade-off平衡关系。

> **注：** $ |\Delta \phi| < \pi $的限定，本质是「结合目标实际运动速度，排除了“相位差加减$ 2\pi $”的实际可能性」，并非单纯从数学上否定“加减$ 2\pi $的理论存在”，而是实际应用中，这种情况不会发生，因此能确保相位差唯一对应目标速度，避免测速模糊。
>

### 3.雷达测角
毫米波雷达测方位角和俯仰角原理：通过多个天线阵列接收回波信号，利用相位差或时间差计算目标的方位角和俯仰角。

水平双天线算左右方位角，垂直双天线算上下俯仰角；靠同一回波到达两天线的相位差值，结合波长直接解算空间角度。

小间距密排天线阵列 → 相位差测角  
大间距分布式雷达矩阵 → 时间差测角

相位差计算目标方位角/俯仰角$ \theta $的公式为：

$ \theta = \arcsin\left(\frac{\Delta \phi \cdot \lambda}{2\pi d}\right) $

其中，$ \Delta \phi $ 是相邻天线接收的回波信号的相位差，$ \lambda $ 是雷达发射电磁波的波长，$ d $ 是天线阵列中相邻天线之间的距离。

时间差计算目标方位角/俯仰角$ \theta $的公式为：

$ \theta = \arcsin\left(\frac{\Delta t \cdot c}{
2d}\right) $

其中，$ \Delta t $ 是相邻天线接收的回波信号的时间差，$ c $ 是电磁波在空气中的传播速度(光速)，$ d $ 是天线阵列中相邻天线之间的距离。

目标高度$ h $的计算公式为：

$ h = R \cdot \sin(\phi) $

其中，$ h $ 是目标的高度，$ R $ 是目标与雷达之间的距离，$ \phi $ 是目标的俯仰角。

#### 角度分辨率
角度分辨率$ \Delta \theta $是雷达能够区分两个目标的最小角度差，计算公式为：

$ \Delta \theta = \frac{\lambda}{d} $

其中，$ \lambda $ 是雷达发射电磁波的波长，$ d $ 是天线阵列中相邻天线之间的距离。天线间距越 大，角度分辨率越高。

### 4.雷达方程
雷达方程描述了雷达系统的性能，表达式为：

$ P_r = \frac{P_t G_t G_r \lambda^2 \sigma}{(4\pi)^3 R^4 L} $

其中，$ P_r $ 是雷达接收的回波功率，$ P_t $ 是雷达发射的功率，$ G_t $ 和 $ G_r $ 分别是雷达发射和接收天线的增益，$ \lambda $ 是雷达发射电磁波的波长，$ \sigma $ 是目标的雷达散射截面，$ R $ 是目标与雷达之间的距离，$ L $ 是系统损耗。

## 二、毫米波雷达系统构成
### 1.毫米波雷达的硬件构成
毫米波雷达系统主要由以下几个部分组成：

+ 发射端：发射天线（阵列设计）、振荡器（产生Chirp信号）、功率放大器
+ 接收端：接收天线（阵列）、混频器、滤波器、ADC（模数转换）
+ 信号处理单元：FPGA/MCU/ARM/DSP（核心计算，处理相位差、拍频等信号）

### 2.毫米波雷达的链路处理流程
核心流程：信号发射→回波接收→混频滤波→AD转换→参数解算（距离/速度/角度）→目标识别→数据输出

+ 信号发射：发射机产生毫米波信号(电磁波)，并通过天线阵列将其发射到空间中。
+ 回波接收：当电信号遇到物体发生漫反射，部分信号原路反射回雷达系统，接收机通过天线阵列接收这些回波信号，并将其转换为电信号。
+ 混频滤波：将接收到的高频信号与本地振荡器产生的参考信号进行混频，得到一个较低频率的中频信号，便于后续的处理。
+ AD转换与信号处理：通过ADC（模数转换）将模拟中频信号转换为数字信号，再利用核心信号处理算法（FFT快速傅里叶变换解算拍频/多普勒频率、CFAR恒虚警检测过滤干扰），提取目标的距离、速度、角度信息，完成参数解算。
+ 数据处理：对提取的目标信息进行进一步处理，如目标跟踪、分类和识别等，提升目标信息的准确性和可用性。
+ 数据输出：将处理后的目标信息通过数据接口传输到后续的决策和控制系统，支持自动驾驶、无人机避障等应用场景的实现。

## 三、FMCW雷达的完整信号处理流程
### 1.信号发射与接收
#### 1.1 发射信号生成
FMCW雷达的发射信号是线性调频连续波（Chirp信号），其瞬时频率随时间线性变化：

$ 
 f(t) = f_0 + \frac{B}{T}t
 $

其中，$ f_0 $ 是起始频率，$ B $ 是信号带宽，$ T $ 是Chirp持续时间。

#### 1.2 信号发射
+ 发射天线阵列将Chirp信号辐射到空间
+ 信号以光速传播，遇到目标后发生反射

#### 1.3 回波接收
+ 接收天线阵列捕获反射回波
+ 回波信号包含目标的距离、速度和角度信息

### 2.混频与中频处理
#### 2.1 混频过程
将接收到的回波信号与本地振荡器产生的参考信号进行混频，得到中频信号：

$ 
 IF(t) = A \cdot \cos(2\pi f_b t + \phi)
 $

其中，$ f_b $ 是拍频（包含距离信息），$ \phi $ 是相位（包含速度和角度信息）。

#### 2.2 滤波与放大
+ 低通滤波器滤除高频分量
+ 中频放大器提升信号幅度，满足ADC输入要求

### 3.数字信号处理
#### 3.1 模数转换（ADC）
将模拟中频信号转换为数字信号，采样率需满足奈奎斯特采样定理：

$ 
 f_s > 2f_{b,max}
 $

#### 3.2 距离FFT（1D-FFT）
对每个Chirp的采样数据进行快速傅里叶变换，得到距离维频谱，峰值位置对应目标距离。

#### 3.3 多普勒FFT（2D-FFT）
对多个Chirp的同一距离单元数据进行FFT，得到速度维频谱，峰值位置对应目标速度。

#### 3.4 角度估计（DBF/MUSIC）
+ **数字波束形成（DBF）**：通过天线阵列加权求和，形成指向不同方向的波束
+ **MUSIC算法**：利用子空间分解，实现超分辨角度估计

### 4.目标检测与参数估计
#### 4.1 CFAR检测
恒虚警率（CFAR）检测算法用于从噪声中提取目标：

+ 单元平均CFAR（CA-CFAR）
+ 有序统计CFAR（OS-CFAR）
+ 自适应CFAR

#### 4.2 参数解算
+ **距离**：从距离FFT峰值位置计算
+ **速度**：从多普勒FFT峰值位置计算
+ **角度**：从角度估计结果提取

#### 4.3 目标跟踪
利用卡尔曼滤波或粒子滤波对目标进行跟踪，预测下一时刻目标位置，提高检测稳定性。
