# 2.1 数据集类型概述

## RGB图像
RGB图像是红（Red）、绿（Green）、蓝（Blue）3个颜色RGB图像是红(Red)、绿(Green)、蓝(Blue) 3个颜色通道相互叠加从而得到的彩色图像,这是目前应用最广泛的颜色定义标准。

## RGB-D图像
RGB-D图像使用深度相机进行采集，深度相机技术相当成熟，成本低廉。RGB-D图像在RGB色彩信息的基础上加入了深度（Depth）信息。Depth Map（深度图）是包含与视点的场景对象的表面的距离有关的信息的图像或图像通道。其中，Depth Map 类似于灰度图像，只是它的每个像素值是传感器距离物体的实际距离，RGB图像和与之对应的深度图之间的像素点具有一一对应关系，两种图相结合，即为RGB-D图像，图像中的每个像素都含有颜色信息和深度信息。

## 点云数据
点云数据（point cloud data）是指在一个三维坐标系统中的一组向量的集合，由激光雷达生成。扫描资料以点的形式记录，每一个点包含有三维坐标，有些可能含有颜色信息（RGB）或反射强度信息（Intensity）。具有稀疏性和不规则形。
